Droplist
含义: 下拉框字典；数量: 0-n
属性名称含义可用值默认值
全部适用id字典的ID号串(无)
treelistURL 或 treelist下拉Treelist的URLURL串(绝对或相对)(无)
注: 如果为空，表示是dropList(即最简单的下拉)；否则就是dropTreeList
dataURL数据的URL数据的URL(绝对或相对)(无)
注: 如果为空，dropTreeList自动采用自身的dataURL、 dropList则采用下面跟随的item条目
dataCol绑定的下拉数据的"数据列"的列名列名(无)
displayCol绑定的下拉数据的"显示列"的列名列名，其内容用于显示(无)
注: 允许是以“=”开头的表达式, 可绑定多列(服务器搜索除外), 可参见树列表的演示页"29.下拉的其它功能"
用于dropListisSortByKey下拉内容是否按Key重新排序true/falsefalse
用于dropTreeListisMultiSelectAble是否允许作多行选择true/falsefalse
MultiSelectSequence多选的返回内容的排序方式check - 按鼠标选取的时间先后排序;
row - 按行号排序;
data - 按数据排序;
text - 按文字内容排序;		check
isAutoSelectChildren是否自动选择树杈的子孙true/falsefalse
注: 仅用于isMultiSelectAble=true 时
isTreeNodeSelectAble树杈是否允许被选中true/falsefalse
作用范围1: 单列多层树
作用范围2: 多列多层树在多选时(checkbox勾选)
expandTree如果输入框尚未输入内容, 首次下拉时是否自动展开(Expand)其树杈all - 展开全部树杈;
first - 展开第一个树杈;
Level (n) - 展开前 n 层, 例如 "Level1";
no - 不展开;		no
用于editAbledropTreeListRefreshDelay服务器搜索时, 键盘触发刷新的最短时间间隔毫秒数, 须大于100毫秒600
RefreshChars服务器搜索时, 键盘触发刷新的最短字数字数1
dropdownMode下拉的显示方式auto - 在输入过程中会自动下拉;
buttonClick - 只有在鼠标点击右侧按钮时才下拉;		auto
buttonMode右侧按钮的行为auto - 点击时触发下拉动作;
sendMessage - 只触发"Clicked"事件(p4参数为"button")，不触发下拉动作;		auto
用于DropTreelist 和 editAbleDropTreeListDropdownButton自定义下拉时弹出窗口下方的按钮请参见下方的备注3见备注3

备注1: 如果treelistURL、dataURL是相对URL，应该是相对于当前Treelist XML，而不是相对于页面;
备注2: 如果EditAbleDropTreelist、且dataURL是表达式，就表示服务器搜索, RefreshDelay和RefreshChars用于控制搜索的频度;
备注3:自定义按钮的语法为 DropdownButton="属性=值; 属性=值; ... 属性=值" 这样的串，属性和值的规范如下：
属性含义值
Skin 或
buttonSkin		按钮样式windows - 操作系统外观(默认);
flat - 平坦、无边框
ShowText中间的文字串, 常用于分页显示时的某些提示
可以包含宏, 例如："共@Pages页, @totRows行", 宏在显示时会被替换成真实的值, 可用宏的含义如下：
@Pages - 总页数;
@Page - 当前页;
@totalRows - 总行数;
@startRow - 当前页的开始行;
@endRow - 当前页的结束行;
button1.ID第一个按钮的功能ID每个按钮都必须有唯一的ID号，为整数，分为内部ID和自定义ID, 所谓内部ID，表示硕正已经在内部实现了功能的按钮，ID号功能如下：
　　65534 - "清理"按钮
　　65533 - "关闭"按钮
　　65532 - "确认"按钮
　　65531 - "前页"按钮(用于分页的下拉)
　　65530 - "后页"按钮(用于分页的下拉)
自定义功能，ID范围为 1 到 65520, 点击后将触发 “Clicked”事件
button1.Text 或 button1.Title按钮上的文字文字串
button1.img 或
button1.image		按钮上的图标图标的URL, 也可以使用如下几个内置的、固定写法的图标：
!Bmp/Clear.bmp - 默认的"清理"按钮上的图标
!Bmp/Close.bmp - 默认的"关闭"按钮上的图标
!Bmp/Ok.bmp - 默认的"确认"按钮上的图标
!Bmp/Prior.bmp - 默认的"前页"按钮上的图标
!Bmp/Next.bmp - 默认的"后页"按钮上的图标
button1.Tip按钮上的Tip文字文字串
button1.width指定按钮的宽度像素数，如果不指定宽度，将采用实际宽度
button1.minWidth指定按钮的最小宽度, 以便可以动态缩放像素数
button1.align按钮的排列对齐方式可以是 left 或 right
如果有多个自定义按钮，则按照 "button2."、"button3." 这样的前缀命名.
