Pager 　　
含义: 分页器的属性；数量: 0-n
属性名称含义可用值默认值
fontIndex文字字体数字，指向在<Fonts>中定义的字体的顺序号，从0开始计数0
textColor文字颜色颜色串#000000
totalRows数据的总行数数字或URL. 如果是URL, 返回的响应必须是数字(即总行数), 或包含数字的XML/JSON(无)
pageRows每页显示行数数字100
dataURL数据的URLURL，书写规则举例如下：
　　dataURL = "../getData.aspx?count=@rows&sr=@startRow"
或
　　dataURL = "../getData.aspx?count=@rows&sp=@page"
其中3个宏变量含义如下:
@rows - 每页行数, 即本次要求服务器返回的数据行数;
@startRow - 开始行的序号, 从0开始计数;
@Page - 页序号, 从0开始计数;
注1：@startRow 和 @Page 宏二选一即可, 不需要全部都写;
注2：宏变量在实际发送的http请求中会被替换成真实数字的.		(无)
pageRowsList每页显示行数的下拉框 例如:
　　pageRowsList="20行,50行,100行,200行"
标准的格式是:
　　显示的文字[数字],显示的文字[数字],...
如果 “显示的文字” 中包含了数字, 那么 "[数字]" 部分可以忽略, 比如:
　　pageRowsList="20行,50行,100行,200行,全部显示[1000000]" 		(无)
showText文字串，常用于显示当前行号和行数带宏的串, 例如："共@Pages页, 第@startRow至@endRow行, 共@totRows行", 里面"@"开头的是宏, 宏在显示时会被替换成真实的值, 可用宏的含义如下：
@Pages - 总页数;
@Page - 当前页;
@totalRows - 总行数;
@startRow - 当前页的开始行;
@endRow - 当前页的结束行;
　　如果 showText 设成空串表示不需要显示该文字		(@startRow-@endRow) / @totalRows
layout界面内容布局逗号分隔的串, 可以带 first、 pagecode(或纯数字)、 last、 input、 pageRowsList、 showText, 布局是按照其先后次序从左往右展开的。 其含义分别为：
First - 需要首页按钮(超链接);
Last - 需要末页按钮(超链接);
pagecode - 尽可能多地展现出页码按钮(超链接)。 注: 如果设了 pagecode, 就不能再设下面的纯数字了;
纯数字 - 最多只展现 n 个页码按钮(超链接)。 注：如果设定数字，就不需要前面的 pagecode 了;
input - 可直接输入页码的那个小输入框;
pageRowsList - 每页显示行数的下拉框, 注：必须要先定义 "pageRowsList" 属性，否则设了也没用;
showText - 文字串;
　　如果是按钮样式( style="pagecode=button" ), 还可以有如下2个内容选项:
Prior - 需要上页按钮;
Next - 需要下页按钮;		first, pagecode, last, input, pageRowsList, showText
style显示的样式多个"名=值"的串, 例如："pagecode=button;prior=前页;next=下页", 可用的名值规定如下：
pagecode - 页码的样式，可以为 hyperLink 或 button, "hyperLink"表示超链接样式, "button"表示按钮样式, 默认是hyperLink ;
　　如果是超链接样式，就不需要设其它属性了。 按钮样式可以有如下内容选项:
BackColor 或 BgColor - 按钮的颜色;
BorderColor - 按钮边框的颜色, 不指定就无边框;
CurrentBackColor 或 CurrentBgColor - 当前页按钮的颜色;
CurrentBorderColor - 当前页按钮边框的颜色;
CurrentTextColor - 当前页的页码文字颜色;
First - “首页”按钮上的文字;
Prior - “上页”按钮上的文字;
Next - “下页”按钮上的文字;
Last - “末页”按钮上的文字;
FirstImage - “首页”按钮的图片URL;
PriorImage - “上页”按钮的图片URL;
NextImage - “下页”按钮的图片URL;
LastImage - “末页”按钮的图片URL;
disableAlhpha - 如果按钮采用图片，当图片状态为 Disable 时，灰度的透明度，范围是1-255, 默认是160;		pagecode=hyperlink
备注: showText允许以"|"分隔中英文(用于中英文切换);
