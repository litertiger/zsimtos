Img 或 Image 　　
含义: 图片的属性；数量: 0-n
属性名称含义可用值默认值
src 或 URL图片的URL，支持jpg、bmp、png、gif、tif、ico格式URL串, 支持图片扩展URL(无)
isTransparent是否透明，即是否以SRCAND画图true/falsefalse
isUseOriSize是否以原始尺寸显示true/falsefalse
arrange当图片尺寸和矩形框不相等时，图片的显示排列方式tile - 平铺;
stretch - 伸展;
(left/center/right) 和 (top/middle/bottom)的组合 - 单幅原样显示，水平、垂直的对齐组合, 例如"bottom,right";		(无)
注: 仅用于 isUseOriSize 为 false 时
alpha透明度0 - 255255
gammaGamma校正1.0表示正常亮度, 小于1变亮，大于1变暗1.0
rotate顺时针旋转一个角度0 - 360, 顺时针旋转0
flip翻转Vert - 垂直翻转;
Horz - 水平翻转;		(无)
isGray是否以黑白(灰度)显示true/falsefalse
transparentColor指定图片的透明色颜色串(无)
pathId使用路径<Paths>中某个Path资源，表示肯定画在该封闭Path内，不太常用(无)
isBindingCol是否数据绑定列true/falsefalse
注: 用于函数GetChangedXML()生成的XML串、以及 Load( ) 加载的数据
