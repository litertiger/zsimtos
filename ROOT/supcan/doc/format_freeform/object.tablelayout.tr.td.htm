TD
含义: 和 html 的 TD 同义，TD的子节点就是具体的内嵌对象；数量: 1-n
属性名称含义可用值默认值
width列宽数字, 和<col>中的"width"同义，优先级高于<col>，但不建议书写在此(无)
border-left-width 或 leftBorder左边界线宽度像素数, 对相邻单元格的同一位置的线条也有效(无)
border-right-width 或 rightBorder右边界线宽度像素数, 对相邻单元格的同一位置的线条也有效(无)
border-top-width 或 topBorder上边界线宽度像素数, 对相邻单元格的同一位置的线条也有效(无)
border-bottom-width 或 bottomBorder下边界线宽度像素数, 对相邻单元格的同一位置的线条也有效(无)
bgColor 或 backColor填充的颜色颜色串(无)
备注: 在freeform设计器的工具条中，有边界线色和填充色的辅助功能按钮;
