Text
含义: 纯文字对象的属性；数量: 0-n, 内容: 文字
属性名称含义可用值默认值
常规text 或 value文字串(无)
fontIndex文字字体数字，指向在<Fonts>中定义的字体的顺序号，从0开始计数0
color 或 textColor文字颜色颜色串#000000
bgColor 或 backColor底色颜色串，不定义底色表示透明(无)
align文字水平对齐left / center / rightleft
vAlign文字垂直对齐top / middle / bottomtop
displayMask显示掩码表达式和Treelist的显示掩码类似，但仅支持文字、文字颜色这二种子表达式(无)
isMultiLine 或 isMultiLineAble如果超出宽度, 是否允许自动拆分成多行显示true/falsefalse
tip 或 toolTip鼠标滑过文字时的黄底文字串(无)
isHyperLink是否超链接true/falsefalse
isBindingCol是否数据绑定列，用于函数GetChangedXML()生成的XML串true/falsefalse
textId指定用于显示的数据列的列名(出现在数据的XML中)串(无)
