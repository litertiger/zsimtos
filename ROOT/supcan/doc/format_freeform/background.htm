Background 　　
含义: 背景属性；数量: 0-1
属性名称含义可用值默认值
bgColor 或 backColor背景色颜色串，也可以是以逗号分隔的2个颜色(渐变填充)(无)
colorAngle渐变背景色的旋转角度0 - 360, 顺时针旋转0
Picture 或 backPicture 或 bgPicture背景图片URL串, 支持图片扩展URL(无)
arrange背景图片显示方式tile - 平铺;
stretch - 伸展;
(left/center/right) 和 (top/middle/bottom)的组合 - 单幅原样显示，水平、垂直的对齐组合, 例如"bottom,right";		tile
rotate顺时针旋转一个角度0 - 360, 顺时针旋转0
flip翻转Vert - 垂直翻转;
Horz - 水平翻转;		(无)
alpha透明度0 - 255， 透明度越低(即数字越小)，图像越淡.255
gammaGamma校正1.0表示正常亮度, 用于调校颜色，不太常用1.0
isGray是否以黑白(灰度)显示true/falsefalse
size背景图缩放宽度、高度，以逗号分隔(无)
transparentColor指定图片的透明色颜色串(无)
pathId使用路径id串, 指向<Paths>中某个Path资源，表示肯定画在该封闭Path内，不太常用(无)
ref引用外部XML描述文件外部XML文件的URL, 可以是相对URL, 该文件须包含<Background>内容, 实现类似页面CSS的统一置换功能(无)
注: 如果本XML文档和外部的XML文档同时定义了某个属性, 则优先使用本文档中定义的属性值
