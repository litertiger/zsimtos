ImageFlow 或 ImgFlow 　　
含义: 图片流的属性；数量: 0-n
属性名称含义可用值默认值
style图片流样式ipod / horzipod
src 或 URL图片流的XML描述文件的URL绝对或相对URL(无)
该XML通常为多记录的树, 其文档规则说明如下：
 1. 每记录可有id、url(或src)、tip(或text)元素;
 2. 其含义为ID、图片URL、文字说明，其中id、url为必需;
 3. url可以是相对URL，相对于图片流XML文档位置;
 4. 除此以外的其它元素为自定义元素，内容可以通过 GetImageProp 或 GetImagePropById 函数获取;
 5. 可参考freeform的demo页实例
slider滚动条的样式scroll - 普通的滑竿;
digital - 数字框;
no - 无滚动条;		scroll
alpha倒影的上部透明度0 - 25570
shadowHeightPercent倒影高度百分比0 - 10018
shadowDistance图片底边和倒影顶部之间的距离(像素)0 - 1001
preLoadImages预加载图片数数字，例如50表示预加载当前中心位置的前、后各50幅50
isToolTipEnabled是否随鼠标显示Tiptrue/falsetrue
autoFlowInterval自动滚动播放的时间间隔数字，单位为秒，0表示不自动滚动0
ScrollTimer鼠标拖拽后的屏幕滚动刷新时间间隔3到200之间的数字，单位为毫秒，设置得太大界面会有明显的闪烁，太小则会增加CPU负荷30
Horz样式distance图片之间的左右相隔数字，像素数20
imageSize指定图片显示的宽和高逗号分隔的数字或百分比(相对于ImageFlow区)200,200
hasMouseButton鼠标划过左、右边界时，是否要浮现前页、下页的三角形按钮true/falsefalse
enlargeActive当前活动图片，放大显示的像素数数字，像素数4
ActiveFocusImage当前活动图片的焦点，通常为一透明的 png 格式文件png 文件的 URL(无)
ActiveFocusSize当前活动图片的焦点大小逗号分隔的数字(无)
Digital滚动条sliderCount显示多少个数字数字, 0表示自动0
sliderPos数字区位于ImaleFlow中的位置topLeft / bottomRight / bottomLeft / topRightbottomRight
sliderColor数字区的底色Html颜色串#C00000
备注: ImageFlow是没有背景属性的，它将沿用当前Freeform的上下文背景;
