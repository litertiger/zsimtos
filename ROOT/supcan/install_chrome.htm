Chrome内核浏览器如何安装硕正套件

常见的 Chrome 内核浏览器的插件安装方法

插件下载处为：supcan.crx

安装说明
　　大部分国产浏览器是 IE/Chrome 双核的, 其中 “兼容模式” 是 IE 内核, “极速模式” 则表示 Chrome 内核。 极速模式下的安装方法大同小异：点击上面的插件链接，让其下载, 然后将文件拖拽到页面中, 就完成了插件的安装.

　　而 Google 自己的 Chrome 浏览器需要 35 版前的版本, 安装方法为: 点击插件, 让其下载, 在右侧 “工具” 菜单栏 ，打开 "扩展程序" ， 直接将 supcan.crx 拖入"扩展程序"页面空白区.
