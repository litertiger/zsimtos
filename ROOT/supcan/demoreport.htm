    　关于硕正报表
    
　　如何能不用编程就让实施人员、最终用
户随心所欲地编排报表、引用数据、打印报
表、设计票据套打，一直是程序员们的梦想。

　　Supcan Report (硕正报表) 使这一梦
想成真。和Excel 等电子表不同的是，硕正
报表侧重于面向企业应用开发，侧重于和企
业的Web应用产品的集成。

　　Treelist、Freeform主要用途是实现对
数据的增、删、改操作，而 Report 则仅仅
是展现、输出数据。

　　Supcan Report具备报表的一般性功
能：任意输入、复制粘帖、Undo、字体、
对齐、颜色、更改单元格样式、组合单元
格、浮动的文字、图片、斜线、统计图、
内置公式函数、表达式解析、自动计算、
文字超宽处理、分类汇总、交叉表、筛选
过滤、定制分页、多工作表等等。

　　Supcan Report有超强的打印输出功
能，例如分页、套打、缩放、精确打印等，
也可以所见所得地将报表转换成PDF、
JPG、PNG、Excel、HTML等格式的文件。

　　从中心数据源取数填充，是硕正报表最
重要的功能。按照硕正公司数据与展现分
离的理念，程序员的任务只是将可能会被
 "使用" 的数据编入中心数据源目录、实现被
取数的功能即可，至于如何被使用、如何展
现，已不必非要让程序员来做。

　　Supcan Report 也允许从其它的XML,
JSON，Web Service，HTML Table等数
据源取数。

　　和 Supcan Treelist、Freeform 一样
，Supcan Report 的存储格式也是XML，
程序员用Notepad也能作一些简单的改动。

　　此外，硕正报表提供了自定义工具条、
自定义菜单以及丰富的API。
